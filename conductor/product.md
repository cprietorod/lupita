# Initial Concept
Luppita is an AI property management agent for Colombian landlords. It uses Google Sheets as a database and provides a natural language interface to query data about rent, contracts, maintenance, and taxes.

## Target Audience
- **Primary Persona:** Independent Landlords - Individual owners managing a few properties directly.

## Primary Goals & Outcomes
- **Save Admin Time:** Automating routine checks like rent collection, contract expiration, tax deadlines (Predial, Retefuente), and IPC rent increases.

## Key Differentiators
- **Natural Language Interface:** Allowing landlords to chat with the agent to query data without complex UIs (e.g., "¿Qué arrendatarios no han pagado este mes?").
- **Google Sheets Backend:** Familiar and accessible data storage.
- **Colombian Real Estate Logic:** Built-in support for local tax calendars, UVT, and IPC.

## Future Expansions
- **WhatsApp Bot:** Allowing landlords to interact with Luppita directly via WhatsApp.
- **Email Reminders:** Automatically notifying tenants of due rent or expiring contracts.
